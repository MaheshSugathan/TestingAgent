# Quick Start: Testing with GitHub Copilot

## 🚀 One-Liner Tests

### Health Check
```
Check API health
```

### Single Evaluation
```
Evaluate: "What is RAG?"
```

### Multi-Turn Test
```
Test conversation: "What is ML?" then "Give an example"
```

## 📋 Common Test Patterns

### Pattern 1: Quick Smoke Test
```
Smoke test: health check + evaluate "What is RAG?"
```

### Pattern 2: Full Evaluation
```
Full eval: "What is machine learning?" - show all metrics
```

### Pattern 3: Batch Test
```
Batch: ["What is RAG?", "How does it work?", "What are benefits?"]
```

## 💡 Pro Tips

1. **Be specific:** "Evaluate and show faithfulness score"
2. **Ask for help:** "What's a good score?"
3. **Get recommendations:** "Should I improve anything?"
4. **Track tests:** "Evaluate with session ID test-001"

## 🎯 Example Workflow

```
1. "Check health"
2. "Evaluate: What is RAG?"
3. "Are scores good?"
4. "What should I improve?"
```
